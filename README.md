# Simple-Math-Quiz

## Description
This program is a simple math quiz written in Java.

The user chooses how many addition problems they would like to solve. The program generates random addition questions, checks the user's answers, calculates their score percentage, and requires a score of at least 70% to pass.

## Features
- User chooses the number of questions
- Randomly generated addition problems
- Tracks correct answers
- Calculates percentage score
- Displays pass or fail message
- Repeats until the user scores 70% or higher

## Technologies Used
- Java
- JOptionPane
- NetBeans
- JDK 26
